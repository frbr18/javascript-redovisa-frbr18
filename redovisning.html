Kmom04

Hur kändes det att bekanta sig med begreppet array?

Innan man använde sig utav array blev det att använda flera variabler som ofta var likvärdiga. Nu när man har en array kan man samla dem i ett ställe. En array är som en låda med likvärdig data eller olika sorters data. För att hämta ett specifikt data använder man något som heter index. Index kan man jämföra med att ha en adress för att veta vilken plats i arrayen som en viss data finns. Nackdelen med att använda sig utav index, är att det inte finns någon ledtråd vad som finns på just den platsen förutom namnet på arrayen. Istället när man använder sig utav en variabel har man ett bra namn som kännertecknar var för data det är till exempel "totalSum". Om man har likvärdig data som vi använt oss i kursmomentet att vi har likvärdig data kan man använda sig utav loopar för att manipulera eller skriva ut datan i arrayen. Arrayer är mycket bra om man ska till exempel skriva ut listor utav data i hemsidan.

Hur kändes det att bekanta sig med begreppet objekt?

Innan man började lära sig om objekt var det som att ha delar utav en Ikea möbel som repensenterar alla variabler. Nu när man använder sig utav objekt är som att ha en fördigsnickrat möbel ifrån Ikea där variablerna har ett sammanhang. Metoderna i objektet är vad ett objekt kan göra, i programmeringsvärlden kan man skriva ut något på skärmen medans i verklighheten kan man jämföra det med att ett bord kan höjas eller sänkas. Objekter har en bra struktur som blir lätt att använda och hitta vad som gör vad. Om det saknas något som man kommer på senare kan man enkelt lägga till det i objektet. När man skapar ett objekt kan man se det som en ritning för att skapa en objekt. Medans när man får en objekt att leka med behöver man använda ordet new ellet Objekt.create(objektnamn).
