Kmom01

Vilken utvecklingsmiljö använder du (berätta om något förändrats sen sist, om du svarat på frågan i tidigare kurser)?

Som IDE använder jag utav Visual studio Code som jag tycker fungerar bra, XAMPP för webb server och för terminalen använder jag cygwin och använder chrome för att testa hemsidan. Så det är inga ändringar ifrån den förra kursen.

Är du bekant med HTML och CSS tillsammans med JavaScript sedan tidigare?

Ja jag har haft en kurs i javascript och dom så jag har testat på det innan men det var ett bra tag sedan så det är skönt att fäscha upp lite gammla minnen.

Gick det bra med JSFiddle/CodePen, några funderingar kring verktygen?

Det tror jag det gjorde använde det inte så mycket förutom att testa att kopiera in min kod i den och det gav inga felmeddelande och den visa samma hemsida som när man använder xampp. Det kanske kommer mer i framtida kmom att man har mer användning för det just nu ser ingen anledning att använda det.

Gick det bra att komma i gång med kursmomentet, var det lagom, för litet, för stort, något som var svårt eller saknades?

Allt gick bra och det verkar var en bra start om man är helt nybörjare som det är meningen. För mig kan det varit lite lätt men skönt med en mjukstart.

Vilken är din TIL för detta kmom?

Tror jag inte har någon speciell TIL för kunde nog det mesta redan dock inte utantill så min TIL får bli att jag kan mer utantill nu.

Kmom02

Har du programmerat i JavaScript tidigare, eller kan du relatera till andra språk du har programmerat i?

Ja jag har testat lite javascript innan denna kurs, har testat lite javascript biblotek som angular och react. Annars gjorde vi en kurs i php som är rätt likt javscript där syntaxen är väldigt lika som if else och looparna. Samma gäller python där syntaxen är lika men har en liten annorlunda sak som till exempel i python använder man inte semicolon istället använder man utav tab för att särskilja koden.

Vad tycker du är det svåraste, mest utmanande i utveckling med HTML, CSS och JavaScript - så här långt?

Det som är det svåra är kombinationen mellan html och js hur man kan använda javascript för att skapa html kod eller ändra på stylen med javascript. Annars var det lite klurigt att med key och mouse events hur de fungerade men tror jag har fått en bra förståelse hur det fungerar.

Hur gick det att utföra respektive uppgift, vilken tog mest tid och vilken var mest lärorik?

Det gick bra att utföra dem. Med hjälp utav videon om hur man skapade en flagga var mest lärorikt tyckte jag. Den visade på ett mycket bra sätt hur positionerningen fungerade i css med absolute och relativ position. Det gjorde att jag kunde utföra uppgiften utan någon stor motgång. Med hjälp utav guiden för att keypress event gjorde det lätt men lärorikt att lösa sandbox2 uppgiften den förklarade enkelt vad alla delar gör så man fick en bra förståelse vad alla delar gjorde. Annars finns det många guider där de bara visar hur man gör utan någon förklaring hur allt fungerar.

Vilken är din TIL för det här kursmomentet?

Min TIL är hur man använder positionering i CSS med hjälp utav relativ och absolut postion. Har stött på det ofta innan men aldrig fått en riktig bra förståelse hur de fungerade men med hjälp utav videon fick en bra förståelse till funktionen.

Kmom03

Hur känner du för funktioner i JavaScript, kan du relatera liknande konstruktioner i något annat programspråk?

Jag gillar funktioner det gör det enklare att struktuera upp koden så man vet vart man har alla delarna i koden. Med bra namn på funktionerna blir det enkelt att veta vad de gör samma med namnen på parameterna så man vet vad som ska skickas med till funktionen. Vi har använt funktioner i programmeringsspråken python och php. I python kallar man dem moduler om jag inte minns fel, dock använd dem på samma sätt som i javascript.

Vad tyckte du om de olika uppgifterna? Hur tänkte du när du utförde dem? Var de svåra eller lätta?

Flag och tärningsuppgiften blev väldigt enkel, tack vare de bra föreläsningarna som visade hur man skulle gå till väga. Annars hade man nog fått klura mer själv och använda google som hjälp eller kolla in guiderna. Med videorna minskade arbetstiden drastikt för mig. Labben tyckte jag var enkel men endå väldigt bra för denna kmoment, den blir enkel om har ett bra humm hur funktioner fungerar. Flagg och tärnings uppgiften var väldigt lärorika där man fick jobba mycket att att ta data ifrån html och bearbeta det i javascript för att sedan skriva ut det. Mina tankar var att göra som kraven säger och utföra dem på det bästa sätt jag kan.

Vad är din TIL för det här kursmomentet?

Min TIL är hur man enkelt kan hämta htmldata i javascript för att bearbeta det för att senare skriva ut det i hemsidan. Med andra ord sammanbertet mellan html och javascript.
